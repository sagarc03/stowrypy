# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-15

### Added

- `StowryClient` for generating presigned URLs
- `presign_get()` method for GET request URLs
- `presign_put()` method for PUT request URLs
- `presign_delete()` method for DELETE request URLs
- `generate_presigned_url()` S3-compatible interface
- Full type hints with py.typed marker
- 100% test coverage
